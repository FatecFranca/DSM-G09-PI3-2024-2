## Orientações para rodar o App Web na máquina

1. Clonar o projeto
2. Executar no terminal 'cd front-end':
     npm install
     npm start
3. Executar no terminal 'cd back-end':
     npm install
     npx prisma init (se necessário)
     npm run dev
4. A página será aberta em seu navegador.
